Weft — explorations logo

Trois directions distinctes. Chaque ligne est le même SVG rendu à 16 px (favicon), 64 px (icône d'application) et 256 px (grand format) — sur fond clair puis sur fond navy, pour vérifier que le mark tient sur les deux thèmes.

Recommandation : direction A — trame sur chaîne. Lecture la plus directe du nom weft (la trame est littéralement le fil horizontal du tissage), géométrie pure qui reste lisible à 16 px en favicon, et zéro ambiguïté de marque avec d'autres projets cloud / VM. B est un letter-mark plus conventionnel ; C est plus dynamique mais perd la lecture "tissage" sous 32 px.
